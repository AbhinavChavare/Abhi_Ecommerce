
# Abhi_Ecommerce

An Ecommerce Website  for branded products

## Screens
- Landing page
- About pge
- Product Listing Page
- Contact Page
- Cart management Page

## Features
- Mobile responsive
- Add to Cart
- Delete from Cart
- Filter by category, brandname, and price.
- Sort by Price and alphabetical order
- Search product
- Fullscreen mode
- Mobile responsive

## Build with
- React Router v6
- useReducer and useContext
- Custom Context Hooks
- useState,UseEffect
- Axios, async, await
- Localstorage
